# myhtml-project
#  Web Development Practicals – 15 & 16

This repository contains the implementation of **Practical 15 and Practical 16**, developed using HTML and CSS. The objective of these practicals is to demonstrate fundamental web design concepts, including layout structuring, image integration, and external stylesheet usage.

---

##  Overview

The project consists of two static web pages:

* **Practical 15**: Displays a collection of butterfly images along with a navigation button linking to a registration page.
* **Practical 16**: Showcases a structured image gallery featuring food images with a clean layout.

Both pages follow a consistent design pattern with **header, content, and footer sections**.

---

##  Key Features

* Structured and semantic HTML layout
* External CSS for styling and separation of concerns
* Image gallery implementation
* Basic navigation using hyperlinks
* Reusable design components (header & footer)
* Beginner-friendly and easy to understand

##  Getting Started

To run the project locally:

1. Clone or download this repository
2. Ensure the folder structure remains unchanged
3. Open the desired file in a web browser:

   * `HTML/Practical15.html`
   * `HTML/Practical16.html`

---

##  Important Notes

* Verify that all image paths are correctly linked (`../assets/Image/`)
* Ensure CSS files are properly referenced (`../CSS/`)
* The navigation link in Practical 15 requires `Practical14.html` to be present
